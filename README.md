### Mel-frequency cepstrum

Compare two audio files.

### Download

* `git clone https://github.com/akinariobi/sound-rec.git`
* `cd sound-rec`

### Install & Run

* `pip install -r requirements.txt`
* `python test.py` (from `src` folder)
